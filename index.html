What do I need to bring?

  • Social Security Card - for you and all dependents
  • Picture ID
  • The following common tax forms:
    • W2
    • 1099- INT, DIV, R, MISC
    • Any other tax document given to you
  • A copy of last year's return, if available
  • Bank account routing number and account number
  • To file a joint return, both spouses must be present
